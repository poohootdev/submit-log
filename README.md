## 😀 서밋로그 생성기

- 서밋로그 문자열 생성기를 만들어 보자!
- 2022-05-17

### 📍현재

- 메모장에 따로 저장한 서밋로그 양식을 복사, 편집하여 서밋

### 📍이슈

- 처음 서밋 할 때, 뭔가 막막.
- 메모장에서 폼을 드래그하여 복사하는 것도 일.
- 사람마다 양식이 달라 통일성이 없다.
- 조직 정책이 변경되었을 때, 능동적으로 대처가 어려울 수도 있다.

### 📍해결 방안

- 서밋로그가 입력폼에 맞게 생성되는 웹페이지 개발

### 📍 기술

- 코어 : React, TypeScript, Next.js
- 스타일링 : TailwindCSS
